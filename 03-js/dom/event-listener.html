Event Listener

Event listener adalah fungsi yang kita daftarkan, ketika suatu kejadian terjadi di halaman web.

Contoh button ini. Ketika user melakukan klik pada button ini, maka akan muncul alert.

Alert ini bisa muncul karena kita mendaftarkan fungsi callback pada event `click` di element button tsb.

Contoh lain adalah form ini.

Ketika user click button submit, event `submit` pada form akan terjadi. Kita bisa mendaftarkan fungsi callback pada event ini untuk melakukan suatu aksi.
