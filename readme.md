## Tujuan
- Pemula bisa menggunakan teknologi web: HTML, CSS, JS, dan NodeJS
- Pemula bisa menggunakan teknologi database MySQL.
- Pemula bisa membuat web sederhana dengan teknologi-teknologi tersebut
